Detailed Description

A 3x3 matrix class for 3D mathematical operations.

This class provides functionality for manipulating 3x3 matrices, which are commonly used in 3D graphics, physics simulations, and linear algebra operations. It supports all standard matrix operations including addition, subtraction, multiplication, determinant calculation, inverse, transpose, etc.

The matrix is stored in row-major order, meaning elements are arranged as: | m00 m01 m02 | | m10 m11 m12 | | m20 m21 m22 |

Common uses include representing 3D rotations, scaling, shearing, and projections.

float Math::Matrix3D::Determinant ( ) const

Calculates the determinant of the matrix.

The determinant is calculated as: det = m00 * (m11 * m22 - m12 * m21) - m01 * (m10 * m22 - m12 * m20) + m02 * (m10 * m21 - m11 * m20)
